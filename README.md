# Portfolio Project - 100Hires Application
## Công cụ đã cài đặt
- Cursor IDE (từ cursor.com)
- Extension **Claude Code** (login bằng tài khoản Claude)
- Extension **Codex – OpenAI’s coding agent** (login bằng tài khoản ChatGPT)
## Các bước đã hoàn thành
Tạo tài khoản GitHub
Tải và cài đặt Cursor IDE
Cài đặt 2 extensions: Claude Code + Codex
Tạo public GitHub repository
Mở repository trong Cursor
Chỉnh sửa file README.md
Commit và Push code lên GitHub
## Khó khăn gặp phải & Cách khắc phục
- Không tìm thấy extension Codex → Search “OpenAI” thay vì chỉ search “Codex” thì mới hiện.
- Lỗi “Cannot read properties of undefined (reading 'activeEditor')” → Mở file README.md trước rồi mới mở Codex, sau đó restart Cursor.
- Ban đầu chưa quen giao diện Cursor → Search YouTube “Cursor IDE tutorial” để làm theo.
